# Datasets

Codigo canonico para tareas de datasets del proyecto.

## Contenido

- `setup/`: descarga de HaGRID y generacion de CSV MST reales.
- `manos/`: sintesis de tonos, QC y balanceo por bloques MST.

## Uso

Ejecuta estos scripts desde la raiz del repositorio con `uv run python ...`.

Consulta [README-Datasets.md](../../README-Datasets.md) para el orden exacto del flujo y los comandos completos.
