# Training

Artefactos del entrenamiento ST-GCN.

## Contenido

- manifiestos de entrenamiento para ST-GCN
- historial de entrenamiento
- checkpoints del modelo supervisado
- logs de entrenamiento

## Origen

Estos archivos se generan a partir de las secuencias preprocesadas y del entrenamiento supervisado o auto-supervisado.

## Uso

Usar estos ficheros para reproducir el entrenamiento, comparar epocas y continuar desde checkpoint.
