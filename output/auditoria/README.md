# Auditoria

Artefactos de auditoria de sesgo y performance.

## Contenido

- resultados de auditoria DPR por muestra
- figuras resumen de accuracy por bloque MST
- TVD canonico por par de bloques

## Origen

Estos archivos salen de `scripts/auditoria/auditoria_dpr.py` y del generador de graficas de auditoria.

## Uso

Usar estos ficheros para reportar el comportamiento por tono MST y documentar posibles disparidades entre bloques.
